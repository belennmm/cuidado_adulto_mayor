## Sobre el proyecto

Organízate es una plataforma web orientada a la administración y monitoreo del cuidado de adultos mayores dentro de casas de retiro.

El sistema permite administrar usuarios, cuidadores, adultos mayores, rutinas, medicamentos, incidentes y turnos desde diferentes tipos de usuario.

---

## Construido con

### Frontend

- HTML5
- CSS3
- JavaScript 
- Boxicons
- Google Fonts

### Backend

- Laravel
- PHP

### Base de datos

- PostgreSQL

### Infraestructura

- Docker
- Docker Compose

---

## Funcionalidades

### Administración

- Gestión de usuarios
- Gestión de adultos mayores
- Gestión de rutinas
- Gestión de turnos
- Dashboard administrativo
- Estadísticas de medicamentos
- Configuración de mensajes globales
- Gestión de solicitudes pendientes

### Cuidador Profesional

- Visualización de adultos mayores asignados
- Gestión de rutinas
- Registro de notas
- Visualización de turnos
- Reporte de incidentes

### Cuidador Familiar

- Visualización del adulto mayor asociado
- Consulta de rutinas
- Consulta de incidentes
- Dashboard personalizado

### Sistema general

- Inicio de sesión por roles
- Registro de usuarios
- Logout con confirmación
- Sidebars dinámicos según el rol
- Estructura preparada para integración completa con backend

---

## Roles del sistema

El sistema cuenta actualmente con tres tipos de usuario:

- Admin	Control total del sistema
- Cuidador Profesional	Manejo operativo y seguimiento
- Cuidador Familiar	Consulta de información del adulto mayor
